# RM 상담차팅 v0.5.3 반영

## 프론트엔드
GitHub Pages `/charting/`에 이미 반영됩니다.

## Apps Script
문단구성 평가와 메모를 Google Sheet의 별도 열에 저장하려면:
1. 기존 Apps Script `Code.gs` 전체를 이 폴더의 `Code.gs`로 교체합니다.
2. 저장합니다.
3. `배포 → 배포 관리 → 수정 → 새 버전 → 배포`를 진행합니다.
4. 기존 `/exec` URL은 유지합니다.

기존 프로젝트 설정값은 그대로 유지합니다.

## 시트
`상담차팅` 탭 끝에 다음 두 열을 사용합니다.
- `문단구성_평가`
- `문단구성_메모`
